1. DESCRIPTION

More Nations Mod adds 166 new nationalities to the game as if they were always there, and rebalances the UnitWeights of all of them (including default nations) to more accurately reflect the present-day military power of each. See the bottom of this file for all nationalities' chances of showing up. All nations have appropriate flag textures and namelists. 118 custom namelists (containing over 26000 names and surnames) are included, with some of the added nations using the default Chinese, Dutch, English, French, German, Greek, Indian, Irish, Italian, and Korean namelists.

------------------
2. COMPATIBILITY

- This mod is compatible with other nation mods, even if they add the same nations, due to MNM countries using a unique identifier. However, you would obviously end up with duplicate nations.
- Mods that alter default nations' UnitWeights (their chance of showing up) will conflict with More Nations Mod, though not in any critical way. See point 6 of this ReadMe for information on how to compatibilise.
- More Nations Mod is compatible with Long War 2, at least at the time of this release.

------------------
3. MANUAL INSTALLATION

To install manually:

1) Copy the contents of the mod package to the directory \Steam\steamapps\common\XCOM 2\XComGame\Mods\

To uninstall, simply delete the MoreNationsMod folder at that location.

6. UNITWEIGHT REBALANCE INFO

UnitWeight is a variable, assigned per nation, that determines the chance of any randomly generated character being of a certain nationality. To edit these UnitWeights, go to Steam\steamapps\workshop\content\268500\619708200\Config\XCOMGameBoard.ini, or to \Steam\steamapps\common\XCOM 2\XComGame\Mods\MoreNationsMod\Config\XCOMGameBoard.ini if not using the Steam Workshop. All .ini files can be worked on in text editors such as Notepad.

To compatibilise with other mods that alter the UnitWeight of default nations, go to the other mods' Config\XCOMGameBoard.ini and either delete the UnitWeights or alter them according to the table below. Otherwise, you might find that characters from the default nations rarely appear, due to More Nations Mod using very inflated values compared to vanilla (this allows for greater granularity).

Chance = (NationUnitWeight/TotalUnitWeight)*100
